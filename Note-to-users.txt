1. The "Format as Text" JSON Content
Tell your users to paste this into the "Content" box of the Bubble "format as text" window:
json
{
  "name": "This Item's Name:find & replace (", \")",
  "quantity": This Item's Quantity,
  "unit_price": This Item's Price,
  "description": "This Item's Description:find & replace (", \")"
}
Use code with caution.

Delimiter: Use a comma ,
Why the find & replace? This is the "Secret Sauce." If a user enters a double quote (e.g., 15" Screen), it will break your JSON. This sanitizes it automatically. [1][3]
